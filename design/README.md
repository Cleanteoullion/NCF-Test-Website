# /design/

Ce dossier pilote la refonte visuelle **v2** du site NCF.

Les fichiers ici sont versionnés avec le code et servent de **source de vérité** pour le design : tokens (couleurs, typo, espacements), composants, références, choix de direction artistique.

## Fichiers

- `design.md` — brief de la refonte v2 (à remplir).
- `clay.md` — système de design Clay.com (cream canvas, cartes saturées, illustrations 3D).
- `mastercard.md` — système de design Mastercard (cream + pills + portraits circulaires).
- `notion.md` — système de design Notion (navy hero, CTA violet, cartes pastel).

Les 3 derniers sont des **références d'inspiration** importées via `getdesign`. Le brief final de la v2 NCF reste à écrire dans `design.md` — il peut piocher dans l'un des 3 ou combiner des éléments.

À enrichir au besoin (`tokens.md`, `components.md`, `refs/` avec captures, etc.).

## Branche

Le travail visuel se fait sur la branche `v2-redesign`. `main` reste sur la v1 actuelle tant que la v2 n'est pas validée et mergée.
